# Tower Travel Tourism Platform

A comprehensive travel booking platform built with Django that allows users to book hotels, flights, and tour packages with integrated payment processing and AI-powered travel assistance.

## 🌟 Features

- **User Authentication** - Login, Signup, Password reset
- **Destination Management** - Browse and search destinations
- **Hotel Booking** - Reserve hotels with date selection
- **Flight Reservations** - Book flights with real-time availability
- **Tour Packages** - Pre-planned travel packages
- **Payment Integration** - Stripe payment processing
- **AI Travel Assistant** - Chatbot for travel recommendations
- **Admin Dashboard** - Manage bookings, users, and content
- **Reviews & Ratings** - User feedback system

## 🛠️ Tech Stack

- **Backend**: Django 6.0
- **Database**: SQLite (Development) / PostgreSQL (Production)
- **Frontend**: HTML5, CSS3, JavaScript, Bootstrap 5
- **Payment**: Stripe API
- **AI**: OpenAI GPT integration for chatbot
- **Deployment**: Ready for Heroku/PythonAnywhere

## 📋 Prerequisites

- Python 3.8+
- pip
- virtualenv (recommended)
- Git
